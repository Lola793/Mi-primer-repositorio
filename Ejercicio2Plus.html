💻 Proyectos de Automatización y Desarrollo

✅ SafeScreen App - Gestión Remota de Dispositivos

SafeScreen es un proyecto personal que permite controlar remotamente dispositivos mediante una aplicación en Flutter, backend en FastAPI con JWT y comunicación segura vía WireGuard VPN en una Raspberry Pi.

🚗 Car Robot IoT - Detección y Activación Automática

Este proyecto personal integra un coche robótico con un sensor PIR y ESP32, conectado a una Raspberry Pi para detectar movimiento y reproducir música automáticamente en un dispositivo Google Nest.

📚 Gestión CRUD Empleados - Java y Spring Boot

Proyecto realizado en una formación parcialmente completada que implementa operaciones CRUD usando Java con Spring Boot y bases de datos SQL.
